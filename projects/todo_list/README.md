# TODO List CLI App

A simple command-line TODO list program built with Python.

## Features

- Allows user to add tasks to the list
- Displays all tasks with numbering
- Shows a message if the list is empty
- Allows user to remove tasks by selecting a task number
- Validates task number before removing a task
- Repeats continuously until the user chooses to exit

## How to Run

Run the Python file:

```bash
python main.py
```

## Example

```
1. Add a task
2. View todo lists
3. Remove a task
4. Exit

Enter the number of the menu(1/2/3/4): 1
Write your todo list: Study Python
Task added
```
