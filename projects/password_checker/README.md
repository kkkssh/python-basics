# Password Checker

A simple Python program that checks whether a password meets basic security requirements.

## Features

- Checks password length and character types
- Provides feedback for missing conditions
- Repeats until a valid password is entered

## How to Run

Run the Python file:

```bash
python main.py
```

## Example

Input:
Password123!

Output:
Success!

## Skills Used

- Variables
- Loops (for, while)
- Conditional statements (if/elif/else)
