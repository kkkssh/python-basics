# Unique Email Filter

A simple Python program that collects email addresses from the user, validates them, removes duplicates, and displays unique results.

## Features

- Accepts user input
- Validates basic email format
- Removes duplicate emails using sets
- Displays total number of unique emails

## How to Run

Run the Python file:

```bash
python main.py
```

## Example

```
Enter email addresses (type 'exit' to finish):
Email: test@gmail.com
Email: hello@gmail.com
Email: test@gmail.com
Email: exit

Unique emails:
test@gmail.com
hello@gmail.com

Total unique emails: 2
```

## Skills Used

- Lists
- Sets (duplicate removal)
- Loops (while)
- Conditional statements (if/else)
- Functions
- String handling (split, strip)
